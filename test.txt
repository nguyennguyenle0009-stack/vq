Cách chạy Gradle Wrapper trên Windows

Mở PowerShell/CMD tại thư mục gốc project (nơi có settings.gradle.kts, gradlew, gradlew.bat, thư mục gradle/).

PowerShell:

.\gradlew clean build
.\gradlew :common:build
.\gradlew :server:run


CMD:

gradlew clean build
gradlew :common:build
gradlew :server:run

Nếu báo “'gradlew' is not recognized…” (wrapper chưa có)

Bạn có 2 lựa chọn:

A) Tạo wrapper (cần cài Gradle tạm thời)

Cài Gradle (hoặc dùng SDKMAN/Chocolatey).

Tại thư mục gốc project, chạy:

gradle wrapper


Sẽ xuất hiện các file:

gradlew
gradlew.bat
gradle/wrapper/gradle-wrapper.jar
gradle/wrapper/gradle-wrapper.properties


Sau đó dùng lại gradlew như hướng dẫn ở trên.

B) Chạy bằng Gradle cài sẵn (không dùng wrapper)
gradle clean build
gradle :server:run

Test nhanh

Mở Postman WebSocket ws://localhost:8080/ws

Gửi JSON:

{"op":"input","seq":1,"press":{"right":true}}


Nhận phản hồi ack và state liên tục.
